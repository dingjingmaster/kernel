# Wayland介绍

Wayland是X11窗口系统协议和体系结构的替代品，其目标是更容易开发、扩展和维护。

Wayland是一种语言(协议)，应用程序可以使用它与显示服务器对话，以使自己可见并从用户(一个人)那里获得输入。Wayland服务器被称为**合成器**。应用程序是Wayland客户端。

Wayland 还指一种系统架构。它不只是合成器与应用程序之间的服务器-客户端关系。与 X11 的 Xorg 不同，Wayland 没有单一通用的服务器，而是每个图形环境都自带众多合成器实现方案中的一种。窗口管理和最终用户体验通常与合成器紧密相关，而非可互换的组件。

Wayland 架构的核心部分是 libwayland：这是一个进程间通信库，它将 XML 格式的协议定义转换为 C 语言的 API。该库并不实现 Wayland，它仅仅对 Wayland 消息进行编码和解码。实际的实现存在于各种合成器和应用程序工具包项目中。

Wayland 并不限制使用位置和方式。Wayland 合成器可以运行在 Linux 内核模式、evdev 输入设备上或某些操作系统上的独立显示服务器，也可以作为 X11 或 Wayland 应用程序（客户端）的嵌套合成器。Wayland 甚至可用于应用程序内部通信，如某些网络浏览器。
