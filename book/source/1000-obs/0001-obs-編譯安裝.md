# obs编译安装

## obs 多个组件

- api(Web API)
- server(构建服务核心)
- client(命令行客户端)
- obs-server(OBS后台服务)
- osc(客户端命令行工具)

## 依赖

- rake

## 环境依赖

- `sudo useradd  -M obsrun`

## 下载 obs 源码

open-build-service

1. 源码根目录执行：
```shell
# 安装 ruby 工具
gem install bundler --no-document -v 2.6.2
```
配置ruby源
```shell
bundle config set mirror.https://rubygems.org https://gems.ruby-china.com
```
配置第三方安装目录
gem config set path 'vendor/bundle'
2. 安装项目依赖
```shell
# 1. src/api/
bundle install -V

# 2. dist/t/services/data/030-bundle_gems/
bundle install -V 

# 3. dist/t/
bundle install -V 
```
