# 磁盘布局

磁盘整体布局如下:
```
+--------+-------+-------+---------+-------+
| 启动块 | 块组0 | 块组1 | ....... | 块组N |
+--------+-------+-------+---------+-------+
```
每个块组布局如下:
```
+------------+-------------+------------+------------+--------+--------+-----------+
| ext4 super | Group       | Reserved   | Data Block | inode  | inode  | Data      |
| Block      | Descriptors | GDT Blocks | Bitmap     | Bitmap | Table  | Blocks    |
+------------+-------------+------------+------------+--------+--------+-----------+
| 1 block    | many blocks | many       | 1 block    | 1      | many   | many more |
|            |             | blocks     |            | block  | blocks | blocks    |
+------------+-------------+------------+------------+--------+--------+-----------+
```
Group 0是一个特殊块, 此块必须保留不使用, 以允许作为x86引导扇区或其用途。

如果将块大小设置为1024，且强制使用block 0，超级块则使用block 1。所以超级块一定不会保存在block 0处。

不是每个块组中都包含 super block, 而是均匀分布在各个块组中(3^0、3^1、5^1、7^1、3^2、5^2、7^2、3^3、5^3、7^3 ...)。
