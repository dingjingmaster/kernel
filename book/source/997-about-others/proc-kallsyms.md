# /proc/kallsyms

## 字段

`/proc/kallsyms` 每行符号信息包含以下部分:

```
地址 类型 名称
```

|类型|含义|
|:---|:---|
|`T`|全局符号, 内核代码段|
|`t`|局部符号, 内核代码段|
|`D`|全局符号, 已初始化数据段|
|`t`|局部符号, 已初始化数据段|
|`B`|全局符号, 未初始化数据段|
|`b`|局部符号, 未初始化数据段|
|`R`|全局符号, 只读数据段|
|`r`|局部符号, 只读数据段|
|`G`|全局符号, 大对象分配区域|
|`g`|局部符号, 大对象分配区域|
|`U`|未定义符号, 外部符号, 未在内核中定义|
